Phase 5 Hardening Features

Task Timeout
Default 10 minutes, configurable per request. Uses AbortController to cleanly terminate long tasks.
Graceful Shutdown
SIGTERM/SIGINT handlers with 30s grace period. Waits for running tasks before exit.
Queue Persistence
Messages have status (pending/processing/completed/failed). Auto-recovery on startup.
Error Classification
Classifies errors as connection_refused, timeout, container_crashed, network_error. Marks recoverable.
Token Tracking
Cumulative input/output tokens per session. GET /tokens endpoint for monitoring.
Rate Limiting
10 messages/minute/agent. Returns 429 with X-RateLimit headers when exceeded.
Log Retention
Configurable via LOG_RETENTION env. Optional file persistence to /ledger/logs.jsonl.
Health Checks
30-second interval checks. 3 consecutive failures = unhealthy. Status in agent list.

Quick Start

# Clone the repo
git clone https://github.com/R-A-V-E-N-delegate/nexus.git
cd nexus

# Set up environment
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env

# Build Docker image
docker build -t nexus-cell:latest cell/

# Start API server
cd api && npm install && npm run dev

# Start Dashboard (new terminal)
cd dashboard && npm install && npm run dev
